**全局层级**：顶部导航栏，提供侧边栏按钮、全局搜索按钮、用户信息按钮、消息通知按钮等功能；导航栏支持页面跳转功能；侧边栏提供

**DashboardPage（仪表盘）**：系统首页 / 概览入口，轻量化，不承担数据管理，只是面向普通用户的内容发现。仪表盘负责展示媒体库统计信息（影片数量、资产数量、标签分布等）、最近导入/最近观看、个性化推荐（后台推荐系统获取数据）。注意，此页面在设计时要考虑可定制化组件添加、数据可视化强化。

**LibraryPage（媒体库浏览）**：用户媒体库入口，按集合展示所有影片（宫格 / 列表切换），支持分类浏览（年份、标签、收藏、资产），支持筛选与排序（评分、时间、标题），用户点击影片可快速跳转到详情页。注意，此页面在设计时要考虑筛选能力强化以及使用虚拟滚动或懒加载技术优化用户体验。

**SearchPage（搜索页）**：检索与智能交互的工作区，接近 webui 界面。搜索页提供两种方式，普通检索 + 语义检索、语义操作（增删改），普通检索用于精确关键词定位，语言检索用于模糊查询、自然语言问题，语义操作负责给用户提供高级的批处理；搜索页提供按钮确认具体的检索方式；搜索页在结果显示上以卡片或列表形式推送检索或修改结果，并支持快速跳转到详情页。搜索页保留检索/操作历史对话记录，支持上下文检索。

**DetailPage（详情页）**：影片档案 + 资产管理的核心页面，详情页采用背景大海报 + 信息区域的结构设计；信息区域负责显示影片 + 资产，其中影片包含影片海报和详细信息（标题、导演、演员、发行时间等）；资产分为官方和用户资产，其中官方资产包含视频、图片、字幕、封面等，用户资产包含截图、笔记、剪辑等；详情页支持跳转播放页、图片放大查看、预览笔记等；详情页支持对资产进行上传/删除/编辑。详情页包含编辑模式，可以对影片信息、资产信息进行修改。详情页包含LLMs接口，支持调用大模型获取影片信息和官方资产。

**DataViewPage（数据总览页）**：管理员/高级用户的数据管理工作台，负责以表格方式展示电影数据、支持高级检索、支持批量操作（如数据导入、批量重命名、批量获取字幕、封面等）。

**SettingsPage（设置页）**：全局系统设置。采用分级设计，避免页面臃肿。设置页包含：界面主题、播放设置、存储路径、数据库配置、模型选择等配置项。

**PlayerPage（播放页）**：独立页面，不受全局组件影响，仅保留一个返回按钮。内置视频播放器：支持倍速、快进、进度条、画中画。包含拓展功能：截图（保存到用户资产）、滤镜（局部应用）等。支持快速笔记功能（书签），供用户记录影片的闪光点。注意，设计时注意优化播放体验与性能，并保留拓展功能插件接口，并设置影片转码接口以适配不同设备和平台。

**AssetsPage（用户资产页）**：用户资产专门处理页。支持浏览 / 管理截图、剪辑、笔记。

**CollectionPage（合集页）**：用户自定义合集入口。用户可创建合集（如「经典科幻」「奥斯卡获奖片」）。支持收藏影片、批量加入。

**用户管理与权限页**：管理员应能在此邀请用户、分配角色（如普通用户、管理员）、设置不同媒体库的访问权限等。开发优先级低。

**任务中心页**：系统涉及批量导入、元数据抓取、视频转码等耗时操作。**任务中心**页面可以集中展示这些后台任务的进度、状态和历史记录，让用户对系统状态一目了然。

**系统状态/监控页** ：此页面可展示服务器的资源使用情况（CPU、内存、存储空间）、API接口健康状况、数据库连接状态等。便于系统维护和故障排查。

**API Playground**：提供一个内置的 API Playground，使用FastAPI自动生成的doc。

**Markdown 专业笔记编辑器**，支持代码块、表格、链接；支持专有的 **影片调用**，如自动插入影片截图、时间戳；支持资产调用，如直接插入用户截图/剪辑，支持发布至三方平台。

**MediaStudio 页面**，在系统内进行轻量化视频编辑，包含了视频剪辑工作台、图像处理实验室、批量处理工具和AI增强功能。
