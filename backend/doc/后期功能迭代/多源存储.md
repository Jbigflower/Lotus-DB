在 资产 Asset & 用户资产 UserAsset 中新增字段：

| 字段             | 含义                                                            | 示例                                                   |
| ---------------- | --------------------------------------------------------------- | ------------------------------------------------------ |
| `storage_type` | 媒体文件的真实存储位置类型（local, smb, s3, webdav, gdrive...） | `"local"` / `"s3"` / `"gdrive"`                  |
| `actual_path`  | 媒体资产真实访问路径                                            | `/mnt/media/movies/Inception.mkv` 或 `https://...` |
| `logic_path`   | 本机逻辑路径，用于缓存 / 缩略图 / 雪碧图等                      | `/data/main/library_1/movies/Inception/`             |

需要额外实现：访问层适配器，定义统一的访问接口，不同存储类型走不同策略。
