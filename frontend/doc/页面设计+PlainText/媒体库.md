## 说明

- AppHeader：右侧全局入口（ LLM 、 检索 、 用户 ）
- BackCrumb：左侧返回与库上下文（ 返回 侧边栏 「媒体库名」详情页 ）
- Toolbar（工具栏）
- StatsIndicator： 总媒体数 / 当前页始 - 当前页终
- RandomEnter：随机进入某媒体详情
- CardStyleToggle：切换卡片样式（紧凑/标准/海报）
- SortSettings：排序字段与方向
- FilterSettings：筛选条件面板
- QuickSearchButton：快速检索入口
- MediaGrid：媒体卡片网格（ MediaCard ）
- PaginationBar：上一页/下一页 + 当前页统计

## 交互与数据

- 加载：进入页面后根据当前库与筛选/排序参数拉取媒体列表
- 分页：点击上一页/下一页更新 page 与 size ；保持筛选条件
- 随机进入：从当前结果集中随机挑选一个媒体跳转详情
- 卡片样式：仅影响视觉呈现，不改变数据
- 排序/筛选：更新查询参数并刷新列表
- 快速检索：跳转至检索页并带入当前库上下文

## 可复用组件映射

- SectionCard（容器）用于工具栏与列表块
- MediaCard/MediaGrid（领域组件）承载电影条目
- Pagination、EmptyState、AsyncState（通用反馈与分页）
- FilterPanel、SortSelect、SearchBar（筛选与检索组合）
- Breadcrumbs/BackButton（导航）
